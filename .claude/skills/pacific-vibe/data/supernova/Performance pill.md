# Performance pill

The Performance Pill is a compact visual indicator that communicates changes in numerical data, such as stock holdings fluctuating in value. Designed for quick comprehension, it provides immediate insight into whether a value has increased, decreased, or remained neutral. The pill can display both dollar ($) and percentage (%) values, using color and subtle iconography to reinforce positive, negative, or neutral changes. By helping users quickly assess performance trends, the Performance Pill enhances data-driven interfaces without overwhelming the UI.

## Accessibility

Assistive technologies, such as screen readers, can read the label on pills, making the meaning of the badge accessible to all users. The label should effectively convey the same message without solely relying on color.

## Usage

While commonly used in financial applications, the Performance Pill can support various contexts where tracking value changes is essential:
Stock market & investments
Displaying portfolio performance (e.g., “+3.5%” or “+$1,200”)
Tracking individual stock price changes
Highlighting daily, weekly, or yearly performance shifts
Transactions
Showing spending trends (e.g., monthly budget usage)
Indicating account balance changes
Tracking interest earned on savings accounts
Price changes
Displaying price fluctuations for products (e.g., “-10%” for a discount)
Showing demand-driven pricing changes
Indicating sales trends (e.g., revenue growth or decline)
