# Selector tile

Selector Tiles offer users clear, scannable options to choose from, presenting relevant and actionable information in a compact and visually appealing format. Key elements such as text and glyphs are strategically placed to ensure a clear indication of hierarchy, enabling users to quickly identify and understand the available choices.

## Accessibility

Once focused, the screen reader should announce all content within the selector tile at once, without requiring navigation between elements.

The screen reader should navigate the page in a logical, hierarchical order, treating the selector tile and its label as a cohesive unit. This ensures that all relevant information is announced together.
If the tiles are part of a group and no selection has been made, the screen reader should navigate them in chronological order. However, if a tile is preselected or a user has previously made a selection, the screen reader should first focus on the selected tile before moving to the others.

Keyboard:
- Tab: Move to the next actionable element on the page
- Space or Enter: Confirm action

## Usage

Should be used in cases where users have to choose from a few given options. Depending on the scenario, users might have the option to select multiple options or only one. These cases should be clearly represented in the prompt's body copy and the appropriate variant should be used.
Positioning
Always place selector tiles on color-surface-base backgrounds. Other surface color usage can be evaluated on a case by case basis.
Clarity
Ensure that text and supporting graphics are intuitive and concise to improve user comprehension and reduce cognitive load.

Grid and layout
Tile Display: Tiles can be displayed in a grid or vertical list.
Avoid Congestion: Organize the grid to avoid congestion or an imbalanced layout.
Responsive layout
Scaling
As tiles scale to adapt to different window size classes, their position and alignment may also change.
Ergonomics
Adjust the layout of components to meet ergonomic needs for large screen devices. For example, a horizontally oriented tile in a large window size may become a vertically oriented group in a compact window size.
Grouping
Tiles may be grouped with other components as long as it adheres to the dimension guidelines and enhances the overall design.
