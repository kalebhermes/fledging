# Global Header

Web Core

The Global Header appears at the top of the page and remains consistent across the product. It contains core navigational elements, brand context, and utility actions, helping users understand where they are, move between sections, and quickly access important functionality regardless of the page they are on.

## Accessibility

globalheader-accessibility-label

globalheader-FocusOrder

Keyboard:
- Tab: Move to the next element
- Space/Enter: Opens link, or opens dropdown menu if available
- Esc: Dismisses dropdown
